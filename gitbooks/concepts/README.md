# Concepts

Core concepts behind Neocortex's memory system. Understanding these will help you get the most out of the API.

- [Memory Items](memory-items.md) — The basic unit of storage
- [Namespaces](namespaces.md) — Organizing memories into logical groups
- [Timestamps](timestamps.md) — Time-aware memory with `created_at` and `updated_at`
- [Ingest](ingest.md) — Storing memories (single and batch upserts)
- [Recall](recall.md) — Retrieving memories with natural-language prompts
- [Context](context.md) — LLM-ready formatted strings from recalled memories
- [Interactions](interactions.md) — Signals that reinforce important memories
- [Memory Decay](memory-decay.md) — Time-decay model inspired by the Ebbinghaus Forgetting Curve
- [Delete](delete.md) — Explicit memory cleanup
